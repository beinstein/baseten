Installing and removing the BaseTen schema

BaseTen Assistant attempts to install the BaseTen schema when entities are enabled or an Xcode data model is imported. There's also a menu command for installing the schema, as well as removing it.

All BaseTen's information is stored in the schema or references objects in it, so removing it will remove elsewhere installed triggers and rules as well.

To install the schema, select File → Install or Upgrade BaseTen Schema.
To remove the schema, select File → Remove BaseTen Schema.
